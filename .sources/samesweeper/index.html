About

Same Sweeper is a mobile puzzle game being developed for Android. It's based on a classic game called SameGame[1] in which you are trying to clean a grid of tiles by removing groups of adjacent tiles with the same color. In Same Sweeper, you can remove selected tiles (and move the rest of the grid) by swiping in any of the four cardinal directions.

There are two main modes in the game —"Random Game" and "Daily Challenge". Random Game is played on randomly generated grid (three different sizes avaialable) and the goal is to achieve highest score possible. Daily Challenge presents a new pre-made level with a specific goal each day.
